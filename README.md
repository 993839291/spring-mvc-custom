# Spring MVC Custom

> A Simple Implementation of Spring MVC

## 介绍

  基于Spring IOC 和 原生Servlet 完成的一个简单的 Spring MVC。

  包含简单处理流程和注解处理流程。

## 项目整体结构

```text

├── adapter           适配器实现
├── annotation        自定义注解实现
├── handler           处理器实现，包含Controller测试
├── ioc               IOC容器实现，基于Spring本身
├── model             ModelAndView
├── servlet           分发器
```
